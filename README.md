# Dark Protocol

> Privacy-focused DeFi protocol with zero-knowledge proofs

## Features

- Private transactions
- Shielded pools
- Anonymous voting
- ZK bridges

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Blockchain: Solidity, Rust, Web3.js
- AI: MiMo V2 Flash, Claude Code

## Installation

```bash
git clone https://github.com/bomniop/dark-protocol.git
cd dark-protocol
npm install
npm run dev
```
